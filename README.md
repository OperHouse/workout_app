# WorkoutApp — Тренировочный дневник для Android

Приложение для планирования тренировок и отслеживания спортивного прогресса. Разработано на языке **Java** с акцентом на работу в offline/online и локальное хранение данных.

## 🚀 Основные возможности
* **Управление пресетами:** Создание и редактирование собственных программ тренировок и приемов пищи.
* **Логирование упражнений:** Удобный интерфейс для записи подходов, весов и повторений в реальном времени.
* **История тренировок:** Просмотр прошедших сессий с детализацией по упражнениям.
*  **История приемов пищи:** Просмотрт записей приемов пищи.
*  **Отслеживание активности и питания:** Отслеживайте количестов набор колорий за сутки и вашу активность при помощи трекеров.
* **Локальная БД:** Полная автономность работы без необходимости подключения к интернету.
* **Подключение к серверу:** Синхронизируте свои данные с сервером.

## 🛠 Стек технологий
* **Язык:** Java (JDK 17+)
* **Архитектура:** MVC / Элементы Clean Architecture
* **База данных:** SQLite (через стандартный Android SQLiteOpenHelper)
* **UI:** XML Layouts, Material Design Components
* **Сборка:** Gradle

## 🛡 Безопасность и данные
Так как проект разрабатывается в рамках изучения информационной безопасности, в приложении реализованы:
* Принципы минимизации привилегий (Permissions).
* Безопасное хранение локальных данных в защищенном песочнице приложения (Internal Storage).
* Отсутствие передачи пользовательских данных на сторонние сервера.
